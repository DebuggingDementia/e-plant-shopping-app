# E-Plant Shopping App 🌱

## Project Description
This is a React + Redux shopping cart application where users can browse plants and add them to a cart.

## Features
- Add products to cart
- Update quantity
- Remove items
- View total price and quantity

## Technologies Used
- React
- Redux Toolkit
- Vite
